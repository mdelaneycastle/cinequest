# CineQuest - Daily Movie Challenge Game Ideas

## 1. **The Matrix - Code Rain Decoder**
*Movie: The Matrix (1999)*
Decode falling green characters to find hidden words before time runs out.

## 2. **Indiana Jones - Temple Pattern Memory**
*Movie: Indiana Jones and the Last Crusade (1989)*
Recreate the "leap of faith" tiles sequence by memorizing and repeating patterns.

## 3. **Die Hard 3 - Water Jug Puzzle**
*Movie: Die Hard with a Vengeance (1995)*
Solve the 3-gallon/5-gallon water jug puzzle to get exactly 4 gallons.

## 4. **National Treasure - Caesar Cipher**
*Movie: National Treasure (2004)*
Decode historical clues using various cipher techniques.

## 5. **Saw - Escape Room Timer**
*Movie: Saw franchise*
Solve logic puzzles before the countdown reaches zero.

## 6. **The Imitation Game - Enigma Decoder**
*Movie: The Imitation Game (2014)*
Simplified version of breaking coded messages.

## 7. **Sherlock - Deduction Grid**
*TV: Sherlock (BBC)*
Use clues to fill in a logic grid puzzle about suspects and evidence.

## 8. **Contact - Prime Number Sequence**
*Movie: Contact (1997)*
Find the pattern in number sequences to decode alien messages.

## 9. **The Da Vinci Code - Anagram Solver**
*Movie: The Da Vinci Code (2006)*
Unscramble anagrams to reveal hidden messages.

## 10. **Inception - Maze Navigator**
*Movie: Inception (2010)*
Navigate through a shifting maze that changes based on your moves.

## 11. **Harry Potter - Potion Mixing**
*Movie: Harry Potter series*
Mix ingredients in correct order and proportions (similar to chemistry puzzle).

## 12. **Mission Impossible - Wire Defusal**
*Movie: Mission Impossible series*
Cut wires in correct sequence based on color/pattern clues.

## 13. **Arrival - Alien Language Pattern**
*Movie: Arrival (2016)*
Identify patterns in circular alien symbols to decode messages.

## 14. **Ready Player One - 80s Trivia Speed Round**
*Movie: Ready Player One (2018)*
Quick-fire multiple choice about 80s pop culture references.

## 15. **Squid Game - Red Light Green Light**
*TV: Squid Game (2021)*
Click/stop at right moments during a timed reaction game.

## 16. **The Prestige - Three Card Monte**
*Movie: The Prestige (2006)*
Track the correct card/object through increasingly complex shuffles.

## 17. **Zodiac - Cryptogram Solver**
*Movie: Zodiac (2007)*
Decode the Zodiac killer's cipher puzzles.

## 18. **Alice in Wonderland - Logic Riddles**
*Movie: Alice in Wonderland (2010)*
Solve the Mad Hatter's paradoxical riddles and logic problems.

## 19. **The Accountant - Number Pattern Recognition**
*Movie: The Accountant (2016)*
Find patterns in financial data/number grids.

## 20. **Westworld - Maze Path Finding**
*TV: Westworld*
Find the correct path through a complex maze with only partial visibility.

---

Each game designed to be completed in 3-5 minutes with varying difficulty levels.
